# Feedback-Collector
A web application to help buisness owners easily collect feedback from their customers/users.

## Stack
### Back-end
+ MongoDB
+ Express.js
+ Node.js
### Front-end
+ React.js
+ Redux.js
### Authentication
+ Google OAuth
+ Passport.js
### Payment
+ Stripe

## Hosting
The application is hosted on [Heroku](https://feedback-collector.herokuapp.com/).
